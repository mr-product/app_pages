What is Secure Entry?

It will display value as a secure entry in the Application and be saved in encrypted form in the application database

How Hide item work?

You can hide or unhide an entry by clicking the hide/show icon on the edit item page. To access a hidden item, you can long hold on the application settings tab.

Want to change Bookmark items order?

You can long hold on bookmak item and drag to your required index position.

Update Locale Database Password

It will change password of locale database and your data will secure by your own password. Please make sure you noted that password for any application support requirenment.
